Ajax笔记

发表于 2016-09-20 | 分类于 JavaWeb |
Ajax

  1. 简介

    Ajax (Asynchronous JavaScript And XML)：异步 JavaScript 和 XML
    不刷新页面向服务器发起请求，通过异步请求方式对页面的局部进行更新

    同步请求：发送请求-等待-发送请求-等待 的请求方式

  1. Ajax 应用关键技术

    1.使用 HTML 和 CSS 构建标准化的展示层；
2.使用 DOM 进行动态显示和交互；
3.使用 XML 和 JSON 等进行数据交换和操纵；
4.使用 XMLHttpRequest 来和服务器进行异步通信；
5.使用 JavaScript 将所有元素绑定在一起；

  2. XMLHttpRequest 

    XMLHttpRequest 实质上是一个 JavaScript 对象，是 Ajax 核心，可以在客户端向服务器发起
HTTP 请求，并且可以访问和处理服务器返回的应答数据；
所有现代浏览器（IE7+、Firefox、Chrome、Safari 以及 Opera）均内建 XMLHttpRequest 对象。

1.XMlHttpRequset 对象创建
    var xhr = new XMLHttpRequest();

2.XMLHttpRequest 对象的属性

    属性                     描述
    readyState                 表示异步请求过程中的各种状态
    onreadystatechange        每次状态改变所触发事件的事件处理程序
    responseText             从服务器进程返回的数据的字符串形式
    responseXML             从服务器进程返回的 XML 文档数据对象
    status                     从服务器返回的响应状态码，比如404(未找到)或200(就绪)
    statusText                 伴随状态码的字符串信息

    1.readyState 属性取值

        0        描述一种未初始化状态：已经创建一个 XMLHttpRequest 对象，但还没有初始化
        1         描述一种待发送状态：此时代码已经调用了 XMLHttpRequest.open() 方法并且已经准备好把一个请求发送到服务器
        2         描述一种发送状态：此时 XMLHttpRequest 对象已经通过 send() 方法把一个请求发送到服务器，但是还没有收到一个响应
        3         描述一种正在接收状态：此时 Ajax Engine 已经接受到 HTTP 响应头部信息，但是消息体部分还没有完全接受结束
        4        描述一种已加载状态：此时，响应被完全接受

    2.onreadystatechange 属性

        onreadystatechange 属性用于存储函数，每当 readyState 属性改变时，被调用；正常情况下被调用 4 次
        通常只需在 readyState 值为 4 时做数据的获取和处理工作

    3.responseText 属性

        readyState 值为 0,1 或 2 时，responseText 包含一个空字符串；
        readyState 值为 3 时，响应中包含客户端还未完成的响应信息；
        readyState 值为 4 时，包含完整响应信息；

    4.status 属性

        200        请求成功
        202     请求被接受但处理未完成
        400     错误请求
        404     请求资源未找到
        500        内部服务器错误

3.XMLHttpRequest 对象的方法

    abort()                        停止当前请求
    getAllResponseHeaders()        获取所有 HTTP 头部，以键/值对形式返回
    getResponseHeader("header")    返回指定 HTTP 头部的串值
    open(method, url)             建立对服务器的调用，method 参数可以是 get,post 或 put,url参数可以是
                                相对 URL 或绝对 URL，这个方法还包括3个可选参数
    send(content)                向服务器发送请求
    setRequestHeader("header","value")
                                把指定请求头设为提供的值，再设置任何请求头之前必须先调用 open() 方法


    1.void open(method, url, asynch, username, password)
        method        :GET,POST,PUT
        url         :所调用资源的 URL,文件在服务器上的位置
        asynch         :true(异步) false(同步)
        username     :指定用户名
        password     :指定密码

            在以下情况中，请使用 POST 请求：
                无法使用缓存文件（更新服务器上的文件或数据库）
                向服务器发送大量数据（POST 没有数据量限制）
                发送包含未知字符的用户输入时，POST 比 GET 更稳定也更可靠
    2.void send(content)
        如果请求为异步的，这个方法就会立即返回，否则它会等待直到接受到响应为止
        可选参数可以是 DOM 对象的实例，输入流或者字符串
        String:仅用于 POST 请求

    3.void setRequestHeader(header,value)
        用于为 HTTP 请求中一个给定的请求头设置值，必须在调用 open() 之后才能调用
        header:要设置的请求名称
        value:要设置的值

    4.string getAllResponseHeaders()

        返回一个字符串，包含 HTTP 请求的所有响应头部，包括 Content-Length,Date 和 URI 等

    5.string getResponseHeader(header)


    使用 Callback 函数
        callback 函数是一种以参数形式传递给另一个函数的函数。

  1. Ajax 应用实现

    1.在页面中定义 Ajax 请求的触发事件
2.创建 XMLHttpRequest 对象
3.确定请求地址和请求参数
4.调用 XMLHttpRequest 对象的 open() 方法建立对服务器的调用
5.通过 XMLHttpRequest 对象的 onreadystatechange 属性指定响应事件处理函数
6.在函数中根据响应状态进行数据获取和数据处理工作
7.调用 XMLHttpRequest 对象的 send() 方法向服务器发出请求
JSON

  1. JSON 是基于 JavaScript 的一种轻量级数据交换格式，数据交换语言。

  2. 结构

    "名/值"对的集合：对象，结构，关联数组

值的有序列表：数组

    对象：
    {
        名:值,
        名:值
    }

    数组：
    [
        value,
        value
    ]

  3. JSON 在 JavaScript 中的使用

    JSON 是 JavaScript 的原生格式，可将一个 JSON 数据赋值给一个 JavaScript 变量

var usersArray =
{
    "users":[
        {"firstName":"saber","lastName":"aser"},
        {"fristName":"archer","lastName":"amiry"},
        {"firstName":"lancer","lastName":"kuqiulin"}
    ]
};
usersArray.users[0].firstName;   //获取
usersArray.users[0].lastName = "ku";  //修改

JavaScript 可以使用 eval() 函数将 JSON 文本转化为 JavaScript 对象
在 eval() 函数中使用额外的圆括号
var text = "{\"province\":\"山东\",\"city\":临沂\"}";
var jsonObj = eval("(" + text + ")");

3.对 Ajax 的实现

    1.ajax() 方法
    2.load() 方法
    3.get() 和 post() 方法
        用于通过 HTTP GET 和 POST 请求从服务器请求数据；
        GET 基本上用于从服务器获得(取回)数据，GET 方法可能返回缓存数据；
        POST 也可用于从服务器获取数据，不会缓存，常用于连同请求一起发送数据；
        get() 和 post() 方法中的回调函数仅在请求成功时可调用；
        如果需要在出错时执行函数，需要使用 $.ajax()

        get() 方法：
            $.get(url,data,callback);

            url:必需参数，规定希望请求的 URL；
            data:可选参数，规定连同请求发送的数据；
            callback:可选参数，请求成功完成时的回调函数；

        post() 方法：
            $.post(url,data,callback);

            url:必需参数，规定希望请求的 URL；
            data:可选参数，规定连同请求发送的数据；
            callback:可选参数，请求成功完成时的回调函数；

        例：$.post("test.jsp",{name:"John",time:"2pm"},
            function(data){
                alert("Data Load:" + data);
            });

    4.getJSON() 方法
        通过 HTTP GET 请求载入 JSON 数据，并尝试将其转为对应的 JavaScript 对象；
        $.getJSON(url,data,callback);
            url:必需参数，规定希望请求的 URL；
            data:可选参数，规定连同请求发送的数据；
            callback:可选参数，请求成功完成时的回调函数；

        例：$.getJSON("test.jsp",{name:"John",time:"2pm"},
            function(json){
                alert("JSON Data:" + json.users[3].name);
            });
